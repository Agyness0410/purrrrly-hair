# 社区特色

我们可以放下手机、放慢脚步，深入思考自己的兴趣、价值观和激情，回归in real life的当下，见天地，见众生，见自己。

社区生活也不是悬浮在当地生态之上的嫁接品，而是以一种新生活共同体与在地发生积极交互，希望对在地产生正面和可持续的影响。
