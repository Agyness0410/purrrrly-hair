# **DNC共享空间**

位处世界文化遗产粤剧博物馆旁，西关“最美老街”，“广州最美骑楼街”的永庆坊里，一个西关大屋的建筑集萃地，粤剧武打行銮舆堂、粤剧行会组织八和会馆、李小龙祖居、詹天佑故居的所在地，镌刻着厚重的文化印记的地方，有一个开放给世界，链接年轻人的空间 - 数字游民社区·广州（DNC）
