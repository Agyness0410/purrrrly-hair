# DN余村

Status: 开放中
中文名: 余村数字游民公社
地址: 浙江省湖州市安吉县天荒坪镇青来集人才社区
简介: “DN余村”全名余村数字游民公社，它是汪洋中的一条船，是各位孤勇者的港湾，也是为数字游民设计建设的一个Safe House。”DN余村“所在的区域，是安吉最受欢迎的旅游和度假目的地之一，2021年与安徽省黟县的西递村一起入选了联合国世界旅游组织“世界最佳旅游乡村”名单。
